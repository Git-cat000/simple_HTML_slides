## 📖 项目简介

`simple_HTML_slides` 是一款纯前端、轻量级的演示文稿模板，基于原生 HTML + CSS + JavaScript 开发，**无任何第三方框架依赖**，主打极简风格与极致易用性。

**🎯 核心亮点：这是一款专为 AI 辅助生成而设计的幻灯片模板。**  
得益于完全开放、语义清晰的 HTML 源码结构，你可以将本模板与你自己的汇报内容文档（`.docx` / `.pdf` / `.md` / `.txt`）一起发送给任意 AI 工具（ChatGPT、豆包、Claude、Kimi 等），通过一条简单的提示词，**在数十秒内即可自动生成一份排版精美、可直接演示的学术汇报文稿**，彻底告别传统幻灯片制作的繁琐手动排版过程。

本模板同时完美适配**学术汇报、课程答辩、课题分享、学术展示**等场景，主要解决传统工具（PPT/Keynote）使用 AI 生成不便等问题：

- 🤖 **AI 原生驱动** —— 开放 HTML 文本结构，为 AI 读取、理解、生成而优化，实现“内容入、幻灯片出”的高效工作流  
- 🚀 **开箱即用** —— 无需学习复杂样式知识，AI 生成后双击 `index.html` 即可演示  
- 📐 **原生 LaTeX 数学公式渲染** —— AI 可自动将文档中的公式转为 LaTeX 格式，完美适配理工科场景  
- 🖼️ **图片一键插入自动排版** —— AI 可智能识别图片引用并嵌入，无需手动调整布局  
- 🧭 **内置目录导航、平滑翻页、页码进度条** —— AI 生成时会自动同步目录跳转链接与页码  
- 🌐 **浏览器直接打开即演示** —— 支持所有现代浏览器，无需安装任何软件  
- 📝 **代码结构清晰、全注释** —— 即使人工修改也极其简单，AI 更能精准定位替换内容  

> 💡 **AI 工作流范例**：将本模板 `index.html` 与你的论文摘要或汇报大纲一同上传给 AI，使用下文提供的提示词模板，AI 将自动填充标题、正文、公式、图片路径，并同步目录页码。你只需保存返回的 HTML 文件，即可获得一份可直接用于汇报的学术幻灯片——全程无需手动排版，效率提升数十倍。当然亦可自己手动编辑。

### 2. 🤖 使用 AI 一键生成演示文稿（强烈推荐）

这是本模板最具价值的应用方式。遵循以下三步，让 AI 成为你的专属幻灯片制作助手：

**第一步：准备材料**
- 本模板的核心文件 `index.html`（或整个模板文件夹）
- 你的汇报内容文档（支持 `.docx`、`.pdf`、`.md`、`.txt` 等任意文本格式）

**第二步：发送给 AI 并粘贴以下提示词模板**（请根据实际需求微调）：

    ```
    请根据我提供的汇报内容文档，使用附件中的 `index.html` 幻灯片模板生成一份学术汇报演示文稿。  
    要求：  
    1. 严格遵循原模板的 HTML 结构和 CSS 样式类，仅替换和填充文本内容，不要改动任何样式定义。  
    2. 保留封面页、目录页、正文页、公式页、图片页、结束页的基本结构，若无对应内容则保留占位。  
    3. 将文档中的章节标题自动映射为幻灯片页面标题（`<h2>`），正文内容填充至对应 `<p>` 标签。  
    4. 若文档中包含数学公式，请转换为 LaTeX 格式并用 `$$ ... $$` 或 `$ ... $` 包裹。  
    5. 若文档中提及图片文件名（如 `fig1.png`），请保留 `<img src="文件名" class="slide-img" alt="描述">` 结构，不要修改图片路径。  
    6. 自动计算并同步更新目录页的跳转链接 `jumpTo(页码)`，确保页码与实际幻灯片顺序一致（从0开始计数）。  
    7. 保持代码整洁，输出完整的 `index.html` 文件内容，不要省略任何部分。  
    ```

**第三步：获取并保存结果**
AI 将返回一份已经填充好内容的完整 HTML 代码。你只需：
1. 复制全部代码
2. 粘贴到项目文件夹的 `index.html` 中（覆盖原文件或另存为新文件）（注意不要忽略 `script.js` 和 `style.css` 文件）
3. 双击打开，即可获得一份专业排版的学术演示文稿

> ✨ **为什么这个工作流如此高效？**  
> 传统幻灯片制作中，排版、公式输入、图片对齐等重复性劳动占据大量时间。而本模板将“内容”与“形式”彻底分离：你只需专注准备文字内容，由 AI 自动完成样式套用与结构生成。尤其适合需要将论文、实验报告、课程设计快速转化为汇报材料的科研工作者、教师和学生。

## 🛠️ 详细自定义配置

无论是人工修改还是指导 AI 精准生成，了解以下配置方法都很有帮助。所有修改仅需编辑 `index.html` 和 `style.css` 两个文件，零基础亦可轻松完成。

### 一、基础文字内容修改

打开 `index.html`，找到对应位置直接替换文字即可。

#### 1. 封面页定制

    <!-- 封面核心内容 -->
    <h1>汇报标题</h1>
    <p class="sub-title">汇报副标题</p>
    <p class="author">单位 | 姓名</p>

#### 2. 目录页定制

修改目录标题与导航内容。**特别注意**：`jumpTo(数字)` 中的数字代表页面索引（从 0 开始计数），AI 生成时会自动计算并修正，人工修改时务必同步。

    <!-- 目录标题 -->
    <h2>Outlines</h2>
    <!-- 目录导航项 -->
    <ul>
      <li><a onclick="jumpTo(2)">研究背景</a></li>
      <li><a onclick="jumpTo(3)">公式演示</a></li>
      <li><a onclick="jumpTo(4)">图片演示</a></li>
    </ul>

#### 3. 正文页定制

    <!-- 页面标题 -->
    <h2>正文标题</h2>
    <!-- 正文内容，可添加多个段落 -->
    <p>正文内容段落1</p>
    <p>正文内容段落2</p>

---

### 二、图片插入与配置（AI 自动处理，人工亦可参照）

#### 1. 图片准备
将 PNG / JPG / SVG 等格式图片放入项目根目录（或子文件夹）。

#### 2. 插入图片语法（自动排版）
在正文下方添加 `<img>` 标签即可，单张或多张均自动垂直排列并左对齐。

    <!-- 单张图片 -->
    <img src="实验流程图.png" class="slide-img" alt="实验流程">

    <!-- 多张图片 -->
    <img src="结果图1.png" class="slide-img" alt="结果对比1">
    <img src="结果图2.png" class="slide-img" alt="结果对比2">

#### 3. 图片特性
- 自动与正文左对齐  
- 自动缩放适配页面大小  
- 自动保留统一间距，无需调整样式  

> **AI 提示**：在提供给 AI 的文档中，只需提及“插入图片 `实验流程图.png`”，AI 便会自动生成上述正确格式的标签。

---

### 三、数学公式编辑（LaTeX 语法，AI 无缝转换）

模板内置 MathJax 公式渲染引擎，支持行内公式 `$...$` 和块级公式 `$$...$$`。

    <!-- 块级公式（推荐） -->
    <p>$$ \nabla \cdot \mathbf{E} = \frac{\rho}{\varepsilon_0} $$</p>

    <!-- 行内公式示例 -->
    <p>质能方程为 $E = mc^2$，其中 $c$ 为光速。</p>

    <!-- 支持所有理工科公式：矩阵、微积分、方程组等 -->

> **AI 提示**：你只需在原始文档中用普通文本或 Word 公式编辑器书写公式，AI 会自动将其转换为正确的 LaTeX 格式并嵌入模板。

---

### 四、样式个性化定制（可选）

打开 `style.css` 文件，可自定义模板视觉效果。**AI 生成时通常无需修改样式，以下供有美化需求的高级用户参考。**

#### 1. 修改主题色
全局替换默认主题色 `#2D6CDF` 为自定义颜色值。

    /* 示例：蓝色 → 墨绿色 */
    从 #2D6CDF 改为 #2E8B57

#### 2. 修改字体
默认：英文 `Times New Roman`，中文 `微软雅黑`，可自由替换。

    font-family: 'Times New Roman', 'Microsoft YaHei', sans-serif;

#### 3. 关闭装饰元素
删除 `index.html` 中的以下标签即可去除背景装饰：

    <div class="decor left-decor"></div>
    <div class="decor bottom-right-decor"></div>

---

### 五、页面管理（新增 / 删除）

#### 1. 新增页面
复制任意 `<section class="slide"> ... </section>` 代码块，粘贴到 `<div class="slides">` 容器内。

#### 2. 删除页面
直接删除不需要的 `<section class="slide">` 代码块。

#### 3. 同步目录
修改目录中 `jumpTo(数字)`，数字对应页面索引（从 0 开始计数）。AI 在生成时会自动处理此步骤，人工修改时务必检查一致性。

---

### 六、高级样式调整（可选）

- **正文缩进**：修改 `.slide p` 的 `padding-left: 2ch` 数值  
- **段落间距**：修改 `.slide p` 的 `margin` 数值  
- **动画速度**：调整 `transition` 时间参数（如 `0.3s` → `0.5s`）  
- **页面边距**：修改 `.slide` 的 `padding` 数值
