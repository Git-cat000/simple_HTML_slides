# simple_HTML_slides
✨ 极简 HTML 学术汇报演示文稿模板 | 开箱即用 · 零依赖 · 专注内容

## 📖 项目简介
simple_HTML_slides 是一款纯前端轻量级演示文稿模板，基于原生 HTML + CSS + JavaScript 开发，无任何第三方框架依赖，主打极简风格与极致易用性。
本模板专为学术汇报、课程答辩、课题分享、学术展示场景设计，解决了传统演示工具（PPT/Keynote）排版繁琐、公式难编辑、跨设备兼容性差等痛点：
- 无需学习复杂的样式设计知识，开箱即用
- 原生支持 LaTeX 数学公式渲染，完美适配理工科场景
- 图片一键插入，自动自适应排版，无需手动调整布局
- 内置目录导航、平滑翻页、页码进度条等实用功能
- 浏览器直接打开即可演示，支持所有现代浏览器
- 代码结构清晰，全注释编写，极易修改和维护

## 🚀 快速开始
### 1. 获取项目模板
你可以通过以下两种方式获取 simple_HTML_slides 模板文件：
#### 方式 1：直接下载（新手推荐）
直接下载项目压缩包，解压到本地任意文件夹即可使用。
#### 方式 2：Git 克隆
打开终端，执行以下命令克隆项目到本地：
git clone https://github.com/你的用户名/simple_HTML_slides.git

### 2. 启动演示文稿
零配置、零安装、零编译！
1. 进入项目根目录
2. 找到 index.html 文件
3. 双击文件，使用任意浏览器（Chrome/Edge/Firefox 等）打开
4. 演示文稿自动加载完成，即可开始使用

### 3. 基础操作方式
模板支持多种翻页/导航方式，操作简单便捷：
- 键盘操作：按下 ← 上一页 / → 下一页（最常用）
- 鼠标操作：点击页面底部的左右按钮切换页面
- 目录导航：在目录页点击对应目录项，一键跳转到目标页面
- 页码查看：页面右上角实时显示 当前页码 / 总页码
- 进度条：页面顶部显示整体演示进度

### 4. 内置 Demo 说明
模板默认包含完整的演示页面，可直接参考学习：
1. 封面页：汇报标题、副标题、作者信息展示
2. 目录页：Outlines 目录导航
3. 正文页：通用内容展示模板
4. 数学公式测试页：LaTeX 公式渲染示例
5. 图片测试页：图片自动排版示例
6. 结束页：感谢观看收尾页

## 🛠️ 详细自定义配置
本模板所有修改位置均有明确标注。

### 一、基础文字内容修改
打开 index.html 文件，直接替换对应位置的文字即可。
#### 1. 封面页自定义
找到第 1 页 slide 模块，修改以下内容：
<h1>汇报标题</h1>                 
<p class="sub-title">汇报副标题</p> 
<p class="author">单位 | 姓名</p>   

#### 2. 目录页自定义
找到第 2 页模块，修改目录标题与目录项：
<h2>Outlines</h2> 
<li><a onclick="jumpTo(2)">内容1</a></li>
<li><a onclick="jumpTo(3)">数学公式测试</a></li>

#### 3. 正文页自定义
修改正文标题与正文内容：
<h2>内容1</h2>        
<p>正文内容1</p>      
<p>正文内容2</p>

---

### 二、图片插入与修改（自动排版）
模板支持单张/多张图片插入，自动垂直排列、自适应大小，无需调整样式。
#### 1. 准备图片
将需要插入的图片（PNG/JPG/SVG 格式）放入项目根目录。
#### 2. 插入图片代码
在对应页面的正文下方，添加图片标签：
<img src="your pictures position" class="slide-img">
<img src="your another pictures position" class="slide-img">

#### 3. 图片规则
- 图片自动与正文左对齐
- 自动缩放大小，不超出页面范围
- 多张图片自动垂直排列，保留统一间距

---

### 三、数学公式添加（LaTeX 语法）
模板内置 MathJax 公式库，直接写 LaTeX 语法即可自动渲染。
#### 1. 块级公式（推荐）
<p>$$ \nabla \cdot \mathbf{E} = \frac{\rho}{\varepsilon_0} $$</p>

#### 2. 支持公式类型
支持所有理工科公式：微积分、矩阵、麦克斯韦方程组、三角函数、求和公式等。

---

### 四、主题样式自定义
打开 style.css 文件，可个性化修改模板样式。
#### 1. 修改主题色
全局替换默认蓝色 #2D6CDF 为你需要的颜色：
#2D6CDF → #28A745

#### 2. 修改字体
默认配置：英文=Times New Roman，中文=微软雅黑，可自由替换：
font-family: 'Times New Roman', 'Microsoft YaHei', sans-serif;

#### 3. 关闭装饰色块
若不需要页面装饰元素，直接删除 index.html 中以下标签：
<div class="decor left-decor"></div>
<div class="decor bottom-right-decor"></div>

---

### 五、新增/删除页面
#### 1. 新增页面
复制任意 <section class="slide"> 代码块，粘贴到 <div class="slides"> 容器内即可。
#### 2. 删除页面
直接删除不需要的 <section class="slide"> 代码块。
#### 3. 同步目录页码
新增/删除页面后，修改目录中 jumpTo(数字)，数字对应页面索引（从 0 开始）。

---

### 六、高级样式调整（可选）
- 调整正文缩进：修改 style.css 中 padding-left: 2ch 数值
- 调整页面间距：修改 margin / padding 数值
- 修改动画速度：调整 transition: 0.5s ease 时间参数
