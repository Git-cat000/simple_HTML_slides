## 📖 项目简介
`simple_HTML_slides` 是一款**纯前端轻量级演示文稿模板**，基于原生 HTML + CSS + JavaScript 开发，无任何第三方框架依赖，主打极简风格与极致易用性。
本模板专为**学术汇报、课程答辩、课题分享、学术展示**场景设计，解决了传统演示工具（PPT/Keynote）排版繁琐、公式难编辑、跨设备兼容性差等痛点：
- 无需学习复杂的样式设计知识，开箱即用
- 原生支持 **LaTeX 数学公式** 渲染，完美适配理工科场景
- 图片一键插入，自动自适应排版，无需手动调整布局
- 内置目录导航、平滑翻页、页码进度条等实用功能
- 浏览器直接打开即可演示，支持所有现代浏览器
- 代码结构清晰，全注释编写，极易修改和维护
